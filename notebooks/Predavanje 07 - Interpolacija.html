Interpolacija#

Uvod#

Pri interpolaciji izhajamo iz tabele (razliƒçnih) vrednosti \(x_i, y_i\):

\(\mathbf{x}\)

\(\mathbf{y}\)

\(x_0\)

\(y_0\)

\(x_1\)

\(y_1\)

\(\dots\)

\(\dots\)

\(x_{n-1}\)

\(y_{n-1}\)

doloƒçiti pa ≈æelimo vmesne vrednosti. ƒåe ≈æelimo doloƒçiti vrednosti zunaj obmoƒçja \(x\) v tabeli, govorimo o ekstrapolaciji.

V okviru interpolacije (angl. interpolation) toƒçke pove≈æemo tako, da predpostavimo neko funkcijo in dodamo pogoj, da funkcija mora potekati skozi podane toƒçke.

Pri aproksimaciji (angl. approximation ali tudi curve fitting) pa predpostavimo funkcijo, ki se ƒçimbolj (glede na izbrani kriterij) prilega podatkom.

Interpolacija s polinomom#

Interpolacija s polinomom se zdi najbolj primerna, saj je enostavna!

Polinom stopnje \(n-1\):

\[ y = a_0\,x^{n-1} +a_1\,x^{n-2} +\cdots + a_{n-2}\,x + a_{n-1}. \]

je definiran z \(n\) konstantami \(a_i\). Da doloƒçimo \(n\) konstant, potrebujemo \(n\) (razliƒçnih) enaƒçb. Za vsak par \(x_i, y_i\) lahko torej zapi≈°emo:

\[ y_i = a_{0}\,x_i^{n-1} +a_{1}\,x_i^{n-2} +\cdots + a_{n-2}\,x_i + a_{n-1}. \]

Ker imamo podanih \(n\) parov, lahko doloƒçimo \(n\) neznanih konstant \(a_i\), ki definirajo polinom stopnje \(n-1\). Sistem \(n\) linearnih enaƒçb lahko zapi≈°emo: $\( \begin{bmatrix} x_{0}^{n-1}&x_{0}^{n-2}&\dots&x_{0}^0\\ x_{1}^{n-1}&x_{1}^{n-2}&\dots&x_{1}^0\\ &&\vdots&&\\ x_{n-1}^{n-1}&x_{n-1}^{n-2}&\dots&x_{n-1}^0\\ \end{bmatrix} \begin{pmatrix} a_{0}\\ a_{1}\\ \vdots\\ a_{n-1} \end{pmatrix}= \begin{pmatrix} y_{0}\\ y_{1}\\ \vdots\\ y_{n-1} \end{pmatrix} \)$

Sistem linearnim enaƒçb zapi≈°emo v obliki:

\[ \mathbf{M}\,\mathbf{a}=\mathbf{b}. \]

Definirajmo matriko koeficientov \(\mathbf{M}\):

M = np.asarray([[_**p for p in reversed(range(len(x)))] for _ in x])

Lagrangeva metoda#

Lagrangeva metoda ne zahteva re≈°evanja sistema enaƒçb in je s stali≈°ƒça ≈°tevila raƒçunskih operacij (nara≈°ƒça sorazmerno z \(n^2\) (vir)) bolj≈°a od predhodno predstavljene polinomske interpolacije (≈°tevilo operacij nara≈°ƒça sorazmerno z \(n^3\)), kjer smo re≈°evali sistem linearnih enaƒçb. Re≈°itev pa je seveda popolnoma enaka!

from IPython.display import YouTubeVideo
YouTubeVideo('c_TbU8XBHsw', width=800, height=300)

Lagrangev interpolacijski polinom stopnje \(n-1\) je definiran kot:

\[ P_{n-1}(x)=\sum_{i=0}^{n-1}y_i\,l_i(x), \]

kjer je \(l_i\) Lagrangev polinom:

\[ l_i(x)=\prod_{j=0, j\ne i}^{n-1} \frac{x-x_j}{x_i-x_j}. \]

Poglejmo si interpolacijo za zgoraj prikazane \(x\) in \(y\) podatke.

Definirajmo najprej Lagrangeve polinome \(l_i(x)=\prod_{j=0, j\ne i}^{n-1} \frac{x-x_j}{x_i-x_j}\):

def lagrange(x, x_int, i):
    """ Vrne vrednosti i-tega Lagrangevega polinoma
    
    x: neodvisna spremenljivka (skalar ali numeriƒçno polje)
    x_int: seznam interpolacijskih toƒçk
    i: indeks polinoma
    """
    Lx = 1.0
    for j in range(len(x_int)):
        if j != i:
            Lx *= (x-x_int[j]) / (x_int[i]-x_int[j])
    return Lx

Opazimo, da ima \(i\)-ti Lagrangev polinom v \(x_i\) vrednost 1, v ostalih podanih toƒçkah pa niƒç!

ƒåe torej Lagrangev polinom za \(i=0\) pomno≈æimo z \(y_0\), bomo pri \(x=x_0\) dobili pravo vrednost, v ostalih interpolacijskih toƒçkah pa niƒç; implementirajmo torej Lagrangev interpolacijski polinom:

\[ P_{n-1}(x)=\sum_{i=0}^{n-1}y_i\,l_i(x), \]
def lagrange_interpolacija(x, x_int, y_int):
    """ Vrne vrednosti Lagrangeve interpolacije
    
    x: neodvisna spremenljivka (skalar ali numeriƒçno polje)
    x_int: abscisa interpolacijskih toƒçk
    y_int: ordinata interpolacijskih toƒçk
    """
    y = 0.
    for i in range(len(x_int)):
        Lx = 1.0
        for j in range(len(x_int)):
            if j != i:
                Lx *= (x-x_int[j]) / (x_int[i]-x_int[j])
        y += y_int[i] * Lx
    return y

Ocena napake#

ƒåe je \(f(x)\) funkcija, ki jo interpoliramo in je \(P_{n-1}(x)\) interpolacijski polinom stopnje \(n-1\), potem se lahko poka≈æe (glejte npr.: Burden, Faires, Burden: Numerical Analysis), da je napaka interpolacije s polinomom:

\[ e=f(x)-P_{n-1}(x)=\frac{f^{(n)}(\xi)}{n!}\,(x-x_0)\,(x-x_1)\,\cdots\,(x-x_{n-1}), \]

kjer je \(f^{(n)}\) odvod funkcije, \(n-1\) stopnja interpolacijskega polinoma in \(\xi\) vrednost na interpoliranem intervalu \([x_0, x_{n-1}]\).

Zgled ocene napake#

Pri interpolaciji ponavadi funkcije \(f(x)\) ne poznamo in napako ocenimo s pomoƒçjo formule:

\[ e=\frac{f^{(n)}(\xi)}{n!}\,(x-x_0)\,(x-x_1)\,\cdots\,(x-x_n). \]

Pri tem vrednost \(\xi\) ni znana; ker je v primeru linearne interpolacije (\(n=2\)) drugi odvod sinusne funkcije (\(f^{(n)}\)) med -1 in +1, velja:

\[ |e|\le\left|\frac{-1}{2!}\,(\pi/4-\pi/6)\,(\pi/4-\pi/3)\right|=\frac{1}{2}\,\frac{\pi}{12}\,\frac{\pi}{12}=\frac{\pi^2}{288}=0,034 \]

Poleg Lagrangeve metode bi si tukaj lahko pogledali ≈°e Newtonovo metodo interpolacije.

Kubiƒçni zlepki so pogost naƒçin interpolacije.

Zahtevamo, da je: \(x_0<x_1< \cdots <x_n\).

Od toƒçke \(x_i\) do \(x_{i+1}\) naj bo zlepek polinom:

\[ f_{i,i+1}(x)= a_{i,3}\,x^3+a_{i,2}\,x^2+a_{i,1}\,x+a_{i,0}, \]

pri ƒçemer so neznane vrednosti konstant \(a_{i,j}\).

ƒåe imamo na primer \(n+1\) toƒçk, potem je treba doloƒçiti \(n\) polinomov.

Celotni zlepek ƒçez \(n+1\) toƒçk je definiran z:

\[\begin{split} f(x) = \left\{ \begin{array}{rcl} f_{0,1}(x); && x\in[x_0, x_1)\\ f_{1,2}(x); && x\in[x_1, x_2)\\ &\vdots&\\ f_{n-1,n}(x); && x\in[x_{n-1}, x_n] \end{array} \right. \end{split}\]

Vsak polinom \(f_{i,i+1}\) je definiran s 4 konstantami \(a_{i,j}\); skupaj torej moramo izraƒçunati \(4n\) konstant \(a_{i,j}\).

Kako doloƒçimo konstante \(a_{i,j}\)?

Za doloƒçitev \(4n\) neznak potrebujemo \(4n\) enaƒçb. Poglejmo si, kako jih dobimo:

  • \(n\) enaƒçb dobimo iz interpolacijskega pogoja: $\(y_i=f_{i,i+1}(x_i),\quad i=0,1,2,\dots,n-1\)$

  • 1 enaƒçbo iz zadnje toƒçke: $\(y_n=f_{n-1,n}(x_n)\)$

  • \(3(n-1)\) enaƒçb dobimo iz pogoja \(C^2\)zveznosti:

\[ \lim_{x\rightarrow x_i^-}f(x)=\lim_{x\rightarrow x_i^+}f(x), \]
\[ \lim_{x\rightarrow x_i^-}f'(x)=\lim_{x\rightarrow x_i^+}f'(x) \]

in

\[ \lim_{x\rightarrow x_i^-}f''(x)=\lim_{x\rightarrow x_i^+}f''(x). \]

Skupaj imamo definiranih \(4n-2\) enaƒçbi, manjkata torej ≈°e dve!

Razliƒçni tipi zlepkov se loƒçijo po tem, kako ti dve enaƒçbi doloƒçimo. V nadaljevanju si bomo pogledali naravne kubiƒçne zlepke.

Naravni kubiƒçni zlepki#

Naravni kubiƒçni zlepki temeljijo na ideji Eulerjevega nosilca:

\[ E\,I\,\frac{\textrm{d}^4y}{\textrm{d}x^4}=q(x), \]

kjer je \(E\) elastiƒçni modul, \(I\) drugi moment preseka in \(q(x)\) zunanja porazdeljena sila. Ker zunanje porazdeljene sile ni (\(q(x)=0\)), velja:

\[ E\,I\,\frac{\textrm{d}^4y}{\textrm{d}x^4}=0. \]

Sledi, da lahko v vsaki toƒçki tanek nosilec popi≈°emo s polinomom tretje stopnje.

\(C^2\) zveznost je zagotovljena v kolikor so vmesne podpore nosilca ƒçlenki (moment zato nima nezvezne spremembe).

Manjkajoƒçi 2 neznanki pri naravnih kubiƒçnih zlepkih doloƒçimo iz pogoja, da je moment na koncih enak niƒç (ƒçlenkasto vpetje):

\[ f''(x_{0})=0\qquad\textrm{in}\qquad f''(x_{n})=0 \]

Izpeljava je natanƒçneje prikazana v knjigi Kiusalaas J: Numerical Methods in Engineering with Python 3, 2013, stran 120 (glejte tudi J. Petri≈°iƒç: Interpolacija, Fakulteta za strojni≈°tvo, 1999); podrobna izpeljava presega namen te knjige.

Tukaj si bomo pogledali samo konƒçni rezultat, ki ga lahko izpeljemo ob zgornjih pogojih. V primeru ekvidistantne delitve \(h=x_{i+1}-x_i\) tako izpeljemo sistem enaƒçb (\(i=1,\dots,n-1\)):

\[ k_{i-1} + 4 k_{i} + k_{i+1} = \frac{6}{h^2} \left(y_{i-1} -2 y_{i} +y_{i+1} \right). \]

kjer je neznanka \(k_i\) drugi odvod odsekovne funkcije \(k_i = f''_{i,i+1}(x_i)\).

Re≈°ljiv sistem enaƒçb dobimo, ƒçe dodamo ≈°e robna pogoja za naravne kubiƒçne zlepke:

\[ k_0=k_n=0. \]

Ko doloƒçimo neznake \(k_i\), jih uporabimo v odsekoma definirani funkciji:

\[ f_{i,i+1}(x)=\frac{k_i}{6}\left(\frac{(x-x_{i+1})^3}{h}-(x-x_{i+1})\,h\right) -\frac{k_{i+1}}{6}\left(\frac{(x-x_{i})^3}{h}-(x-x_{i})\,h\right) +\frac{y_i\,(x-x_{i+1})-y_{i+1}\,(x-x_{i})}{h}. \]

Numeriƒçna implementacija#

Najprej pripravimo funkcijo, katera za podane interpolacijske toƒçke re≈°i sistem linearnih enaƒçb in vrne koeficiente \(k_i\):

def kubicni_zlepki_koeficient(x, y):
    """ Vrne koeficiente kubiƒçnih zlepkov `k`, matriko koeficientov `A` in konstant.
    
    x in y predstavljata seznam znanih vrednosti; x mora biti ekvidistanten.
    """
    n = len(x)
    A = np.zeros((n, n)) # pripravimo matriko koeficientov
    h = x[1]-x[0] # korak h
    for i in range(n):
        if i==0 or i==n-1:
            A[i,i] = 1. # k_0 in k_n sta niƒç zato tukaj damo 1
                       # pri vektorju konstant pa bomo dali 0, k_0 in k_n bosta torej 0
        else:
            A[i, i-1:i+2] = np.asarray([1., 4., 1.])
    b = np.zeros(n)
    b[1:-1] = (6/h**2)*(y[:-2] - 2*y[1:-1] + y[2:]) # desna stran zgornje enaƒçbe
    k = np.linalg.solve(A,b)
    return k, A, b

Nato potrebujemo ≈°e kubiƒçni polinom v doloƒçenem intervalu; implementirajmo izraz:

\[ f_{i,i+1}(x)=\frac{k_i}{6}\left(\frac{(x-x_{i+1})^3}{h}-(x-x_{i+1})\,h\right) -\frac{k_{i+1}}{6}\left(\frac{(x-x_{i})^3}{h}-(x-x_{i})\,h\right) +\frac{y_i\,(x-x_{i+1})-y_{i+1}\,(x-x_{i})}{h} \]
def kubicni_zlepki(k, x, y, x_najdi):
    """ Vrne kubicni zlepek pri delitvi `xint`.
    
    :param k: koeficienti kubiƒçnih zlepkov
    :param x in y: znane vrednosti, x mora biti ekvidistanten
    :param x_najdi: vrednosti kjer ≈æelimo izraƒçunati kubiƒçni zlepek
    """  
    h = x[0] - x[1]
    i = int((x_najdi-x[0])//(-h))
    if i >= len(k)-1:
        i = len(k)-2
    out = ((x_najdi - x[i+1])**3/h - (x_najdi - x[i+1])*h)*k[i]/6.0 \
        - ((x_najdi - x[i])**3/h - (x_najdi - x[i])*h)*k[i+1]/6.0 \
        + (y[i]*(x_najdi - x[i+1]) \
        - y[i+1]*(x_najdi - x[i]))/h
    return out

Izraƒçunamo interpolirane vrednosti:

xint = np.linspace(np.min(x), np.max(x), 50)
yint = np.asarray([kubicni_zlepki(k, x, y, _) for _ in xint])

plt.plot(x, y, 'o', label='Interpolacijske toƒçke')
plt.plot(xint, yint, label='Naravni kubiƒçni zlepek')
plt.legend();
